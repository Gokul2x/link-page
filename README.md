# Minimal Tech-Themed Linktree for Gokul

## How to Deploy

1. Clone or download this folder.
2. Place your files/folders as-is in your repository, ensuring SVGs are in `assets/`.
3. Commit & push to GitHub.
4. In your repository:  
   Go to **Settings → Pages**  
   Set the source to root or `/docs` folder where `index.html` is.
5. Your tech-themed link page will be live at `https://yourusername.github.io/your-repo/`.
6. **Customize links or SVGs as needed.**

## Features

- Pure HTML/CSS, no libraries
- Responsive, mobile-optimized
- SVGs are local, monochrome, copyright-safe

## Live Preview

- Paste the HTML/CSS at [CodeSandbox](https://codesandbox.io/) or [CodePen](https://codepen.io/) for instant preview.
